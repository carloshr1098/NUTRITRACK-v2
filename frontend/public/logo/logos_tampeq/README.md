# Logos Tamaño Pequeño (Optimizados)

Esta carpeta contiene versiones optimizadas de los logos de NutriTrack para uso en PDFs y aplicaciones web.

## Especificaciones Recomendadas:
- **Tamaño máximo**: 150KB
- **Dimensiones**: 150x150px para iconos, 300x150px para logos completos
- **Formato**: PNG-24 con transparencia
- **Uso**: PDFs, emails, aplicaciones embebidas

## Archivos:
- `nutritrack-icon.png` - Icono optimizado (~132KB)
- `nutritrack-logo.png` - Logo completo optimizado (~132KB)

## Notas:
- Estos archivos están optimizados para carga rápida
- Mantienen la calidad visual necesaria
- Compatible con jsPDF y otras librerías
